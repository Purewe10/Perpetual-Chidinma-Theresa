# Sales_Performance_Analysis

## Tools used
- Micrsoft Excel

## Project Overview 
Diamonds Sales KPI Dashboard (Excel)
This project presents an interactive Sales KPI Dashboard built in Microsoft Excel to analyze sales performance for a diamond retail business.
The dashboard provides a high-level executive summary of key business metrics such as:
	•	Number of Customers
	•	Total Sales
	•	Number of Orders
	•	Total Profit

##  Project Objectives

The primary goals of this dashboard were to:
	•	Transform raw transactional data into meaningful business insights
	•	Create a clean and professional KPI-driven layout
	•	Enable interactive filtering by region
  
## Business Insights Derived
	•	Certain regions generate higher revenue but lower profit margins.
	•	Technology products show strong order volume.
	•	Standard shipping dominates total orders.
	•	Profit varies significantly across global markets.
